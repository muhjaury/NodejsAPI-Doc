    Token

    Method "Token" digunakan untuk login sekaligus mendapatkan Token dan Refresh Token.

    • URL
    • Parameter
    • Example
    Method URL
    POST http://localhost:3001/user/login
    Jenis Parameter Wajib Tipe Keterangan
    RAW Username Ya String Username
    RAW Password Ya String Password

    Response

    • Response Sukses
    • Response Gagal
    • Penjelasan Response
    Komponen Tipe Keterangan
    _fail[x] String Komentar keterangan fail
    status Number Kode status
    message String Pesan dari failure
    token String Token untuk menggunakan API
    refreshToken String Token yang digunakan untuk refresh Token apabila sudah expire tanpa login ulang

    Refresh Token

    Method "Refresh Token" digunakan untuk memperbaharui token User tanpa harus login ulang

    • URL
    • Parameter
    • Example
    Method URL
    POST http://localhost:3001/api/generate
    Jenis Parameter Wajib Tipe Keterangan
    HEAD username Ya String Username
    HEAD refresh-key Ya String Refresh Key dari login

    Update Data

    Method "Update" digunakan untuk memperbarui data dari User seperti Username dan Password (Admin Privelege)

    • URL
    • Parameter
    • Example
    Method URL
    PATCH http://localhost:3001/api/update/[:Id User]
    Jenis Parameter Wajib Tipe Keterangan
    HEAD username Ya String Username
    Params (In URL) id Wajib String Id User
    RAW username Wajib String Username
    RAW password Wajib String Password
